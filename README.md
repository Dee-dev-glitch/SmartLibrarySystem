# Smart Library Circulation & Automation System

A Python-based system for managing library operations, including book circulation, member management, and automated notifications for overdue items.

## 🚀 Features
- Book Management (add, search, remove)
- Member Management (register, update, track)
- Circulation (borrow, return, overdue tracking)
- Automation (due date reminders, late return alerts)

## 🛠️ Tech Stack
- Python 3.x
- SQLite (optional for persistence)
- Command-line interface (CLI)
